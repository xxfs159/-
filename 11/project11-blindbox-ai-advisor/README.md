# 盲盒 AI 推荐员

人工智能引论大项目 11：聊天式兴趣画像与个性化盲盒推荐系统。

## 运行方式

直接打开 `index.html` 即可使用。项目为纯前端实现，不需要联网，也不需要安装依赖。

也可以在项目目录运行测试：

```bash
npm test
```

## 已实现功能

- 聊天式偏好采集：用户自然输入预算、IP、风格、颜色、用途、是否接受隐藏款溢价。
- 用户兴趣画像：自动结构化为 IP、风格、颜色、用途、预算、溢价偏好。
- 商品知识库：内置 12 个课程演示用盲盒款式，包含系列、价格、稀有度、颜色、风格、用途、库存和描述。
- 个性化推荐：基于多维标签匹配和预算/溢价约束计算匹配度。
- 推荐解释：每个推荐结果展示命中的偏好、价格适配、溢价风险和购买提醒。
- 对比推荐：Top 5 候选款式按价格、匹配度、溢价风险和核心理由对比。
- 知识库筛选：按风格筛选商品，展示商品信息。
- 推荐报告导出：导出 Markdown 推荐报告，方便放进海报、答辩材料或项目文档。

## 技术方案

- 前端：HTML、CSS、JavaScript。
- 推荐引擎：`engine.js` 实现文本偏好抽取、画像合并、商品打分、推荐解释和 Markdown 导出。
- 匹配逻辑：综合 IP、风格、颜色、用途、预算、隐藏款溢价接受度、库存风险。
- 可视化：商品卡片、兴趣标签、对比表和 SVG 盲盒形象均可离线渲染。

## 演示步骤

1. 打开 `index.html`。
2. 点击顶部“示例偏好”，或粘贴 `samples/persona-prompts.md` 中任一段文本。
3. 查看兴趣画像是否更新。
4. 查看最佳推荐、推荐清单和候选对比。
5. 点击“导出报告”，得到 Markdown 推荐报告。

## 课程展示亮点

- 低门槛：用户像聊天一样表达偏好，不需要填写复杂表单。
- 可解释：不只给结果，还说明为什么推荐以及可能的购买风险。
- 可扩展：商品知识库可替换为爬虫数据或电商数据；推荐引擎可升级为向量检索和 RAG。
- 符合项目要求：覆盖聊天式偏好采集、兴趣画像、商品知识库、向量语义检索的简化模拟、个性化推荐解释和对比推荐。

## 后续增强方向

- 接入真实商品数据源，定期更新价格、库存和系列信息。
- 使用 BGE Embedding 或 MiniLM 做真正的语义向量检索。
