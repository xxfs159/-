# 项目提交清单

## 必交材料

- [ ] 项目代码压缩包
- [ ] README 运行说明
- [ ] 项目报告
- [ ] 汇报 PPT 或 PPT 大纲
- [ ] 海报/易拉宝设计稿
- [ ] 系统运行截图
- [ ] 现场演示准备

## 建议截图

1. 系统首页
2. 课程表导入页面
3. 自然语言任务解析结果
4. 智能学习计划表
5. 每日学习时长柱状图
6. AI 学习建议页面

## 现场演示流程

1. 打开系统首页，说明项目目标。
2. 展示课程表、任务表和空闲时间数据。
3. 输入一句自然语言任务，例如：“人工智能：下周三 23:59 AI大项目报告截止，预计6小时，很重要”。
4. 点击生成计划，展示一周计划。
5. 说明计划如何避开课程时间、如何按优先级安排任务。
6. 展示导出 CSV/Markdown 和 AI 总结功能。
