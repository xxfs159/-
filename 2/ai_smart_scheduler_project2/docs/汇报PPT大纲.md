# 汇报 PPT 大纲：AI 智能课表与学习计划生成系统

## 第1页：标题页

- 项目名称：AI 智能课表与学习计划生成系统
- 课程：人工智能引论
- 小组成员：填写姓名
- 指导教师/助教：填写姓名

## 第2页：项目背景

- 大学生任务多、DDL 多、课程安排复杂
- 手工计划容易遗漏、冲突和拖延
- 需要一个能自动理解任务并生成计划的智能工具

讲解词：
本项目关注大学生日常学习管理问题。我们希望系统不仅记录任务，还能像一个学习规划助手一样，自动分析任务紧急程度，并生成可执行的一周计划。

## 第3页：项目目标

- 导入课程表
- 输入作业/考试/项目 DDL
- 设置个人空闲时间
- 自动生成一周学习计划
- 输出学习建议和风险提醒

## 第4页：系统总体架构

建议画图：

```mermaid
flowchart LR
A[课程表/任务/空闲时间] --> B[NLP任务解析]
B --> C[规划型Agent]
C --> D[一周学习计划]
D --> E[可视化展示/导出]
D --> F[DeepSeek学习建议]
```

## 第5页：自然语言解析模块

- 支持“下周三 23:59”“5月20日 20:00”
- 支持“预计6小时”“很重要”“紧急”
- 输出结构化任务字段
- 体现 NLP 信息抽取能力

## 第6页：规划 Agent 设计

- 输入：任务、DDL、课程占用、空闲时间
- 约束：不能占用上课时间，不能超过 DDL
- 策略：优先级 + 紧急程度排序
- 输出：分块后的学习计划

## 第7页：核心算法

- 时间槽建模
- 课程冲突扣除
- 任务评分：
  - priority * 10 + urgency * 100
- 大任务拆分：
  - 每块 30-120 分钟
- 可用时间不足时提示风险
