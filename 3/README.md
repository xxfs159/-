# AI 写周报机器人

项目 3 完整实现：把零散流水账自动转换为结构化 KPI 周报，并生成周环比可视化与 Markdown 文本，适合复制到飞书、钉钉、企业微信、Notion。

## 运行方式

直接打开 `index.html` 即可使用，无需安装依赖、无需联网。也可以在项目目录执行：

```bash
npm test
```

## 已实现功能

- 流水账输入：支持粘贴文本、Markdown 文件导入、浏览器语音识别写入。
- 信息抽取：自动识别任务、成果、问题、风险、下周计划。
- KPI 结构化：统计任务数、完成数、完成率、延期/阻塞、产出数、问题数、风险数、计划数。
- 周环比展示：导入上周 KPI JSON 后自动计算本周与上周差值。
- 可视化：展示任务分布条形图、环比指标表。
- Markdown 输出：生成正式周报，可复制或下载 `.md` 文件。
- 数据延续：导出本周 KPI JSON，下周可作为“上周 KPI”导入。
- AI 润色衔接：一键复制 DeepSeek/Coze 润色 Prompt，保留人工确认与事实校验。

## 技术方案

- 前端：原生 HTML/CSS/JavaScript，保证低门槛演示与部署。
- 抽取引擎：`engine.js` 基于关键词、显式标签、状态规则和指标统计完成结构化解析。
- 可视化：使用 DOM + CSS 条形图和表格，不依赖第三方库。
- 数据格式：输入为纯文本/Markdown，周环比数据为 JSON，输出为 Markdown。

## 演示流程

1. 打开 `index.html`。
2. 点击“载入示例”或导入 `samples/current_week_log.md`。
3. 点击“生成周报”查看 KPI、图表与 Markdown。
4. 导入 `samples/previous_week.json` 查看环比变化。
5. 点击“复制”或“下载 MD”获取最终周报。
6. 点击“导出 KPI JSON”，保存为下周对比基线。
