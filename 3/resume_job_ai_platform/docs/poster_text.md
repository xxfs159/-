# 海报/易拉宝文案  
## 智能简历优化与岗位推荐平台

### 1. 项目背景

大学生求职时经常面临“简历不知道怎么改、岗位不知道怎么选、面试不知道怎么准备”的问题。本项目基于 RAG 检索增强生成思想，构建智能简历优化与岗位推荐平台，实现从简历解析到岗位推荐、从匹配解释到求职材料生成的一站式求职辅助。

### 2. 项目目标

- 自动解析简历文本，抽取技能、项目和基础信息；
- 构建岗位 JD 知识库，实现岗位语义检索；
- 计算简历与岗位的综合匹配度；
- 输出匹配技能、缺失技能和推荐理由；
- 生成简历优化建议、求职信和面试准备要点。

### 3. 技术路线

```text
简历上传
 → 文本提取
 → 技能抽取
 → 岗位 JD 知识库
 → 向量化检索
 → 综合匹配评分
 → 优化建议生成
 → 可视化展示
```

### 4. 核心方法

- **NLP 简历解析**：提取姓名、邮箱、电话、技能关键词和项目关键词；
- **RAG 岗位匹配**：将岗位 JD 作为知识库，将简历作为 Query，检索最相关岗位；
- **向量化表示**：使用 TF-IDF 字符 n-gram 向量化，后续可替换 BGE embedding；
- **综合评分**：融合语义相似度、技能重合度和求职方向匹配；
- **生成式优化**：输出摘要优化、项目经历优化、求职信和面试要点。

### 5. 创新亮点

1. **可解释推荐**：展示匹配技能和缺失技能，避免黑盒推荐；
2. **求职全流程覆盖**：不只推荐岗位，还生成后续求职材料；
3. **轻量可复现**：普通电脑即可运行，适合课堂现场演示；
4. **可扩展性强**：可接入 BGE、DeepSeek、真实招聘数据和向量数据库。

### 7. 应用价值

- 帮助学生快速定位适合岗位；
- 降低简历修改和求职信撰写成本；
- 为就业指导中心提供智能化辅助工具；
- 可扩展到企业招聘初筛和人才画像分析场景。

### 8. 未来展望

- 接入真实招聘网站岗位数据；
- 使用 BGE embedding 提升语义检索效果；
- 使用 DeepSeek 进行多轮简历润色；
- 增加投递反馈，形成个性化推荐闭环。
