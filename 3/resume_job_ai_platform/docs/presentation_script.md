# 现场汇报稿：智能简历优化与岗位推荐平台

各位老师、同学大家好，我们小组展示的项目是“智能简历优化与岗位推荐平台”。

## 一、项目背景

我们关注的是大学生求职场景。很多同学在准备实习或秋招时，会遇到三个问题：第一，不知道自己的简历适合哪些岗位；第二，不知道简历和岗位 JD 的差距在哪里；第三，针对不同岗位写求职信和准备面试比较耗时。

因此，我们希望做一个 AI 求职辅助工具，帮助用户从简历出发，自动匹配岗位，并生成简历优化建议、求职信和面试准备要点。

## 二、系统功能

我们的系统主要有五个功能。

第一，支持上传简历，格式包括 txt、pdf 和 docx。  
第二，自动解析简历中的基础信息和技能关键词。  
第三，构建岗位 JD 知识库，对岗位信息进行向量化检索。  
第四，输出 Top-K 岗位推荐结果，并展示匹配度、匹配技能和缺失技能。  
第五，针对目标岗位生成简历摘要优化、项目经历优化建议、求职信和面试要点。

## 三、技术路线

系统整体流程是：用户上传简历后，系统先进行文本抽取和清洗，然后抽取技能关键词。接着系统加载岗位 JD 知识库，把简历文本和岗位 JD 都转换成向量表示，计算语义相似度。同时，我们还计算技能重合度和求职方向匹配度，最后得到综合匹配分。

在生成阶段，系统会根据匹配结果生成解释性建议，包括为什么推荐这个岗位、当前简历有哪些优势、还缺哪些能力、应该如何修改项目经历。

## 四、核心算法

我们设计的综合匹配公式是：

综合匹配分 = 0.55 × 语义相似度 + 0.35 × 技能重合度 + 0.10 × 方向匹配度。

其中，语义相似度体现简历和岗位 JD 的整体相关性；技能重合度体现硬技能是否匹配；方向匹配度体现用户自己的求职偏好。

这比单纯关键词匹配更全面，也比纯黑盒模型更容易解释。

## 五、创新点

我们的项目有三个亮点。

第一，可解释。系统不仅给出推荐岗位，还会展示匹配技能和缺失技能。  
第二，覆盖完整求职流程。系统不是只做岗位推荐，还会生成简历优化、求职信和面试要点。  
第三，轻量可复现。即使没有 GPU 或大模型 API，也可以在普通电脑上运行演示；如果配置 DeepSeek API，还可以获得更自然的生成内容。

## 六、演示说明

接下来演示系统。我们使用示例简历，选择求职方向为“大模型应用”，点击开始智能匹配。系统会输出岗位推荐列表，可以看到每个岗位的匹配度、匹配技能和缺失技能。选择一个目标岗位后，系统会生成匹配解释、简历摘要优化、项目经历建议、求职信和面试准备要点。最后还可以下载 Markdown 格式的求职优化报告。

## 七、不足与改进

目前系统仍有一些不足。岗位数据是示例数据，规模还不够大；向量化方法默认使用 TF-IDF，语义理解能力不如 BGE embedding；技能抽取采用词典法，对上下文理解还比较有限。

后续我们计划接入真实招聘网站数据，使用 BGE embedding 和向量数据库提升检索效果，并结合用户投递反馈不断优化推荐结果。

## 八、总结

总的来说，本项目实现了一个完整的 AI 求职辅助流程，涵盖简历解析、岗位知识库、RAG 检索、匹配评分和生成式优化。系统具有较好的可解释性、可复现性和应用价值。我们的汇报到此结束，谢谢大家。
