# 易拉宝/海报内容：CodeCoach-AI 编程题错解诊断与变式训练系统

## 标题

CodeCoach-AI：会跑代码、会讲错因、会出变式题的编程 AI 助教

## 一句话介绍

面向编程学习者的错解诊断系统：输入题目、错误代码和样例，自动运行验证、定位常见错误、生成讲解报告，并推荐同类变式训练。

## 痛点

- OJ 只告诉 Wrong Answer，不告诉为什么错。
- 学生看懂题解后，仍然不会迁移到类似问题。
- 调试过程缺少记录，错误容易反复出现。

## 系统流程

1. 输入题目描述、错解代码、测试样例。
2. 静态分析检查语法、边界、类型和常见坑。
3. 本地沙箱编译/运行代码，对比期望输出。
4. 生成 Markdown 诊断报告。
5. 可选调用 DeepSeek 生成更自然的讲解。
6. 输出变式训练题，形成学习闭环。

## 技术方案

- 前端展示：Gradio
- 代码运行：Python subprocess + g++/Python 本地执行
- 静态分析：Python AST + C++ 规则检查
- 报告生成：Markdown 模板 + DeepSeek API 可选增强
- 训练推荐：基于题目标签的变式题生成

## 创新点

1. 静态分析、动态测试与大模型讲解融合。
2. 从“判断对错”升级为“解释错因 + 推荐训练”。
3. 支持无 API 离线演示，现场稳定性高。
4. 可扩展为个人编程错题本和长期学习画像。

## 展示效果

默认演示题：两数求和。错误代码将 `a+b` 写成 `a-b`。系统自动发现样例失败，输出测试结果表、错误排查顺序和变式训练题。

## 未来方向

- 使用 Clang AST / tree-sitter 做更精确的 C++ 代码分析。
- 使用 Docker 实现更安全的代码运行沙箱。
- 接入 LeetCode / 洛谷题库，建立知识点标签体系。
- 增加错题本、学习周报和个性化训练路线。
