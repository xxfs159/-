本代码库包含多个独立的人工智能与软件开发项目。每个子项目均具有独立的业务场景、依赖文件和说明文档。为了便于阅读和复习，各个子项目的核心逻辑部分均附带有详细的代码注释。

## 目录结构

### 核心项目

* **项目 2：个性化学习 AI 助手 (`2/personal_study_ai_assistant_project`)**
  * **简介**: 记录学习数据并提供个性化进度分析与问答的 AI 系统。
  * **主要组件**: `chatbot.py`, `analysis.py`, 数据库存储模块。

* **项目 4：CodeCoachAI 编程教练 (`4/CodeCoachAI`)**
  * **简介**: 智能编程辅助系统，提供代码逻辑解析、变体分析以及安全的沙盒测试环境。
  * **主要组件**: `analyzer.py`, `sandbox.py`, `reporter.py`。

* **项目 5：AI 简历与求职平台 (`5/resume_job_ai_platform`)**
  * **简介**: 自动解析简历文本，并利用 RAG（检索增强生成）技术与岗位库进行智能匹配的平台。
  * **主要组件**: `resume_parser.py`, `rag_matcher.py`, `generator.py`。

* **项目 11：AI 盲盒顾问 (`11/project11-blindbox-ai-advisor`)**
  * **简介**: 基于 Web 的前端交互式盲盒推荐应用。
  * **主要组件**: HTML 结构、CSS 样式表以及核心驱动逻辑 `app.js` 与 `engine.js`。

* **项目 12：AI 情感树洞 (`12/ai_emotion_treehole_project12`)**
  * **简介**: 具备用户情感倾向分析与心理危机关键词识别预警能力的对话引擎。
  * **主要组件**: `emotion_core.py`, `crisis.py`, `response_generator.py`。

* **项目 39：BongoCat 具身 AI 桌面宠物 (`39/project39-bongocat-ai`)**
  * **简介**: 结合著名 BongoCat 形象的桌面交互式 AI 宠物应用，包含详细的架构与答辩演示方案。
  * **主要组件**: Node.js 服务端 (`server.mjs`) 与前端交互逻辑 (`app.js`)。

* **项目 47：自动化代码安全审计系统 (`47/`)**
  * **简介**: 集成大语言模型（LLM）与 GitHub API 的自动化漏洞扫描、审计与代码补丁生成工具。
  * **主要组件**: `scanner.py`, `github_api.py`, `patcher.py`, `llm.py`。

### 其他测试与示例代码

* **文件夹 1**: 包含用于测试的 React 组件代码（`代码.jsx`）。
* **文件夹 3**: 包含独立的 JS 测试引擎示例（`engine.js` 及配套测试用例）。

## 使用说明

由于各项目的技术栈（涵盖 Python、Node.js 与原生前端代码）与运行环境不同，请在进入具体的子项目目录后，优先查阅该目录专属的 `README.md`、项目报告以及环境配置文件（如 `requirements.txt` 或 `package.json`）以获取完整的启动步骤。建议在阅读代码时结合其中的详细注释，以辅助理解核心模块的运行机制。
